# CEDRI - Artificial Intelligence Manager

This project, developed at the Polytechnic Institute of Bragança (IPB), provides a simplified infrastructure for managing the lifecycle of Artificial Intelligence models through network communication.

- **Communication Management**: Utilizes the `port-manager` library to handle asynchronous TCP connections.
- **Operations Protocol**: Native support for creating, loading, saving, training, and processing models.
- **Data Security**: Integrated validation module to ensure that neural network configurations and hyperparameters are correct before processing.
- **Hybrid Persistence**: Metadata storage in a database and physical model serialization within the file system.
